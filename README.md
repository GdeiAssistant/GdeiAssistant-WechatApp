# 广东二师助手微信小程序

广东二师助手微信小程序客户端，基于微信小程序原生框架与 WeUI 构建，面向广东第二师范学院校园场景，覆盖校园查询、校园生活、消息通知与个人资料等常见功能。

## 项目概览

- 微信小程序原生框架
- WeUI
- Promise 风格服务层
- `mock` / `remote` 双数据源
- i18n 国际化（6 种语言）

## 快速开始

### 运行环境

- 微信开发者工具
- 微信小程序基础库 `>= 2.32.3`
- Node.js `24.14.1`（与 `.nvmrc` 和 CI 保持一致）

### 导入与运行

1. 使用微信开发者工具导入项目根目录。
2. 按需检查 `config/index.js` 中的环境配置。
3. 在微信开发者工具中编译并预览页面。

### 验证命令

```bash
npm ci
npm run lint
npm run format:check
npm test
npm run smoke
```

## 法律、隐私与安全提示

- 本项目是校园助手类小程序，不代表学校官方服务。
- 小程序内的关于、隐私、校园凭证、二手、跑腿、失物招领等提示页面，构成面向用户的主要风险说明入口。
- 生产部署前，部署方应结合实际环境审查校园凭证处理、第三方服务启用情况、日志脱敏、密钥管理、传输加密和商标/名称使用边界。
- 校园凭证相关功能的授权、撤回、删除和快速认证控制依赖后端支持；部署衍生版本前，应确认后端的加密、授权持久化和删除行为已正确配置。
